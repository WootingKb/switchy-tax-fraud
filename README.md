# Gamejam

## Technologies Used

- TypeScript
- React
- Phaser 3
- Vite
- Yarn

## Adding Real Assets

The game currently uses generated placeholder assets. To add your own custom assets:

1. Add your image files to the `public/assets/` directory
2. Update the asset paths in `src/game/scenes/GameScene.ts`
